= linkki
:source-file: ../../../../samples/getting-started/src/main/java/org/linkki/samples/gettingstarted/pmo/ReportPmo.java

Ein Data-Binding Framework für webbasierte Benutzeroberflächen mit https://vaadin.com/home[Vaadin].

== Features

Häufig bestehen Applikationen aus vielen Eingabeelementen, die direkt Objekte und Eigenschaften aus dem Domänenmodell widerspiegeln. Die Erstellung solcher Oberflächen, sowie die Implementierung der Synchronisierung mit den Modellobjekten, ist oft eine repetitive Aufgabe. *linkki* übernimmt die Aufgabe dieses Data-Bindings weitestgehend. So können sich Entwickler auf das Domänenmodell und die UI Logik konzentrieren.

* Schnelle Erstellung von UIs 
* Deklarative UI Definition (über Annotationen)
* Implementierung der UI Logik über ein PresentationModel (PMO)
* Automatisches Data-Binding zwischen PMO und UI Elementen
* Dynamisches Binding weiterer UI Eigenschaften, u.A.
** Sichtbarkeit (visible)
** Aktivierungszustand (enabled)
** Liste der auswählbaren Werte

== Installation

*linkki* steht auf Maven Central zur Verfügung und kann über Maven/Gradle eingebunden werden:

.maven dependency
[source,xml]
----
<dependency>
  <groupId>org.linkki</groupId>
  <artifactId>linkki-core</artifactId>
  <version>${linkki.version}</version>
</dependency>
----

Um *linkki* selbst zu kompilieren, muss das Projekt über https://github.com[github] ausgecheckt werden. Als Build-Tool wird Apache Maven verwendet.
[source,bash]
----
$> mvn clean install
----
